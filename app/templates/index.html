What is this?

This christmas I am giving the masses control of my christmas tree lights. This page is the endpoint from which the lights can be controlled. It contains a real-time image of what the tree is looking like right now along with some controls for changing the LED arrangement and colors.

Any interaction you have with the page here is forwarded down to the christmas tree and you can watch as your commands battle those of other users online right now.

How is it made?

A Raspberry Pi sitting on my desk is connected to a webcam and a custom dongle I built that controls a string of 50 WS2811 individually-addressable LEDs. That Raspberry Pi then SSH's into my cloud server and connects to a Redis instance hosted there. A Flask-based python application on the cloud server connects to the same redis instance and uses the PUBSUB functionality to receive the realtime webcam image published by the Raspberry Pi and also issue commands back to the Raspberry Pi. Socketio is used to communicate between this (static) page and the python app running on the server.
